## Overview

The Employee Leave Tracker is a modern leave management solution that helps organizations efficiently manage employee time off. It provides a centralized platform for employees to request leave, managers to approve/reject requests, and HR to maintain comprehensive leave records.

## Features

### ðŸŽ¯ Core Features

- **Leave Request Management**: Easy-to-use interface for submitting leave requests
- **Multi-level Approval Workflow**: Configurable approval chains for different leave types
- **Leave Balance Tracking**: Real-time visibility into available leave balances
- **Calendar Integration**: Visual representation of team leave schedules
- **Multiple Leave Types**: Support for vacation, sick leave, personal days, and custom leave types
- **Notifications**: Email and in-app notifications for request updates
- **Reporting & Analytics**: Comprehensive reports on leave patterns and trends
- **Mobile Responsive**: Access the system from any device

### ðŸ‘¥ User Roles

- **Employee**: Submit and track leave requests, view leave balance
- **Manager**: Approve/reject team leave requests, view team calendar
- **HR Admin**: Manage leave policies, configure leave types, generate reports
- **Super Admin**: System configuration and user management

## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Valid employee credentials

### Installation

For detailed installation instructions, see [INSTALLATION.md](./INSTALLATION.md)

### Quick Start

1. **Login**: Access the system using your employee credentials
2. **Check Balance**: View your available leave balance on the dashboard
3. **Request Leave**: Click "New Request" and fill in the required details
4. **Track Status**: Monitor your request status in real-time
5. **Receive Notification**: Get notified when your request is processed
